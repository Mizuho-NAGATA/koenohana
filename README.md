# koenohana
鳴き声wavファイルを選択し、つぼみになぞらえた再生ボタンをクリックすると花が開きます。鳴き声の鋭さで花の大きさが変わります。　　
![nakigoenohana](https://github.com/user-attachments/assets/e8b1a215-f013-42ee-9164-6b919eeb5d63)

---

## 使い方（操作手順）
index.html をwebブラウザで開きます。
1. 「ファイル選択」ボタン（またはドラッグ＆ドロップ対応があれば該当部分）から鳴き声の WAV ファイルを選択します。
2. 選択したファイルの鳴き声散布図が画面に表示されます。
3. 再生ボタンをクリックすると、対応する花が開きます。
4. 花の大きさは鳴き声の「鋭さ」（例：高周波成分の強さや音の立���上がり）により変化します。

ヒント：
- wavファイルの数に応じて花が咲きます。
- 録音した愛鳥のwavファイルを、鳴き声ごとの複数の短いファイルに分けて、複数選択すると、たくさんの花がさきます。
